# Program Overview

This program implements a **memory-efficient word-level indexing system** using C++.
It processes large log files and builds an index that maps each unique word to its frequency.

The system supports multiple analytical queries such as:

* Word Count Query
* Top-K Frequent Words Query
* Difference Query between two versions of a dataset

All words are converted to lowercase to ensure **case-insensitive matching**.
For example:

ERROR = Error = error

---

# Object-Oriented Design

The program is designed using **object-oriented programming principles**.

### Classes Used

**Tokenizer**
Responsible for converting words to lowercase for case-insensitive processing.

**BufferedFileReader**
Reads the input file and extracts words from the dataset.

**Indexer**
Builds the word-frequency index using an `unordered_map`.

**Query (Abstract Base Class)**
Defines a common interface for executing queries using a virtual function.

**WordQuery**
Implements the word count query.

**TopKQuery**
Displays the top-K most frequent words in a dataset.

**DiffQuery**
Computes the difference in frequency of a word between two versions of datasets.

---

# Key Concepts Implemented

### Inheritance and Polymorphism

The `Query` class is an abstract base class.
`WordQuery`, `TopKQuery`, and `DiffQuery` inherit from it and implement the `execute()` function.

### Function Overloading

The program demonstrates function overloading using multiple `printResult()` functions with different parameters.

### Templates

A template function `printVector()` is used to demonstrate generic programming.

### Exception Handling

The program uses `try`, `catch`, and `throw` to handle runtime errors such as file opening failures.

# Word-Level Index

The system creates a **word-level index** that maps each unique word to its frequency.

Example:

```
error -> 2
debug -> 1
info -> 1
```

This index is then used to answer queries efficiently.
